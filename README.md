# WooCommerce Quote System

Transform your WooCommerce store into a powerful quote request system by hiding shopping features and adding professional quote forms.

## Description

WooCommerce Quote System is a comprehensive WordPress plugin that converts your WooCommerce store into a quote-based business system. Perfect for B2B businesses, custom manufacturers, service providers, or any business that prefers quote-based sales over direct online purchases.

## Installation

1. Upload the plugin files to `/wp-content/plugins/woocommerce-quote-system/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings ‚Üí WooCommerce Quote System to configure the plugin
4. Set up your form shortcode and customize button text
5. Enable the features you want to use

## Requirements

- WordPress 5.0 or higher
- WooCommerce 5.0 or higher
- PHP 7.4 or higher

## Configuration

### Basic Setup
1. **Enable Plugin**: Master switch to activate all functionality
2. **Form Shortcode**: Enter your form plugin shortcode (e.g., `[contact-form-7 id="123"]`)
3. **Button Text**: Customize the text for quote and WhatsApp buttons

### Advanced Options
- **Hide Price**: Remove all price displays from products
- **Hide Add to Cart**: Replace cart buttons with quote buttons
- **Hide Shopping Elements**: Remove cart, checkout, and related features
- **WhatsApp Integration**: Enable WhatsApp button with your business number
- **Modal Customization**: Set custom titles and subtitles for the quote modal

## Frequently Asked Questions

### Can I use this with any form plugin?
Yes! The plugin works with any form plugin that uses WordPress shortcodes. Simply enter your form's shortcode in the plugin settings.

### Will this affect my existing WooCommerce data?
No, the plugin only hides frontend elements. Your products, orders, and other WooCommerce data remain intact and can be restored by deactivating the plugin.

### Can I customize the appearance?
Yes, the plugin includes comprehensive CSS that you can override in your theme. All elements use the `wcqs-` prefix for easy targeting.

### Does it work on mobile devices?
Absolutely! The plugin is built with a mobile-first approach and includes touch-optimized interactions, swipe gestures, and responsive design.

### Can I enable only some features?
Yes, each feature has its own toggle switch, so you can enable only the functionality you need.
